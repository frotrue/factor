# 🧠 The Neural Factory (가제)

## 1. 게임 개요 (Overview)
**The Neural Factory**는 '공장 자동화(Factorio-like)'와 '타워 디펜스', 그리고 '방치형/인크리멘탈(Incremental)' 장르가 융합된 2D 웹 기반 게임이다. 
플레이어는 물리적인 광석 대신 **'원시 데이터(Raw Data)'**를 채굴하고, 이를 가공하여 **'AI 모델'**을 학습시키는 데이터 파이프라인을 구축해야 한다. 공장의 최종 목표는 끊임없이 밀려오는 악성 데이터(적)를 추론(Inference)하여 필터링하는 방어망을 유지하고, 극한의 스케일로 에포크(Epoch)를 갱신하는 것이다.

## 2. 핵심 게임 루프 (Core Loop)
1. **수집 (Mining):** 맵에서 노이즈가 섞인 원시 데이터(Raw Data)를 채굴한다.
2. **전처리 (Preprocessing):** 데이터를 정규화하여 학습 가능한 라벨링 데이터로 가공한다.
3. **학습 (Training):** 가공된 데이터를 조립 기계(신경망 레이어)에 주입하여 모델을 학습시키고 '가중치(Weight Update)' 아이템을 생산한다.
4. **추론 및 방어 (Inference & Defense):** 생산된 가중치를 컨베이어 벨트로 방어 타워에 공급하여 타워의 '신뢰도(Confidence Score)'를 유지하고 적을 방어한다.
5. **스케일링 (Scaling):** 기하급수적으로 늘어나는 적의 물량과 체력에 맞춰, 컨베이어 병목을 해결하고 데이터 처리량을 무한히 확장한다.

## 3. 주요 개체 (Entities)

### 3.1. 적 (Adversarial Data)
*   **개요:** 메인 서버를 붕괴시키기 위해 침투하는 악성 노이즈 및 바이러스 개체들.
*   **세부 기획:** (추후 추가 예정 - 다양한 공격 패턴, 위장 기믹 등 고려)

### 3.2. 방어 타워 (Inference Models)
*   **개요:** 공장에서 생산된 가중치 데이터를 소모하여 다가오는 적을 필터링하고 요격하는 AI 추론 모델 타워.
*   **세부 기획:** (추후 추가 예정 - 타워별 필요 데이터, 공격 방식, 신뢰도 시스템 등 고려)

## 4. 차별점 및 기술적 특징 (Technical Specs)

### 4.1. 기술 스택
*   **Core Engine:** Vanilla JavaScript + HTML5 Canvas (Phaser 3 프레임워크 활용).
*   **Architecture:** 브라우저 및 Windows on ARM 아키텍처 환경에서 네이티브 엔진 없이 가볍고 최고 효율의 성능을 낼 수 있도록 렌더링과 틱(Tick) 연산 분리.
*   **Deployment:** 웹 기반 초기 개발 후, Capacitor를 활용한 모바일(Android/OneStore 등) 플랫폼 패키징 워크플로우 적용.
