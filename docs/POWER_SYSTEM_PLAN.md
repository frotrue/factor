# ⚡ Neural Factory: 전력 시스템 구현 보고서 (Power System Plan)

## 1. 전력 시스템 설계 방향 (Design Approach)

전력 시스템은 게임의 난이도와 확장성을 결정하는 중요한 요소입니다. 본 프로젝트에서는 너무 복잡한 전선 연결 방식(Wire-routing) 대신, **노드 기반 범위형 전력망(Area-of-Effect Power Grid)** 방식을 제안합니다.

### 🔋 핵심 구성 요소
1. **동력원 (Energy Sources)**:
   - 맵에 생성되는 `ENERGY` 매립지 위에 설치하여 전력을 생산하는 **발전소(Power Plant)**.
2. **전력 송신탑 (Power Node)**:
   - 스스로 전력을 만들지는 않지만, 주변(예: 5x5 타일)의 건물들에게 전력을 공급하는 역할.
   - 메인 서버(Core)와 전력 송신탑들이 서로 이어져야 전력망이 형성됨 (그래프/네트워크 구조).
3. **전력 소비처 (Consumers)**:
   - 채굴기(Miner), 데이터 가공소(Processor), 그리고 추후 추가될 **방어 타워(Defense Towers)** 등은 작동 시 전력을 소비합니다.
   - 전력이 부족하거나 끊기면 건물 위에 ⚡(전원 부족) 경고 아이콘이 뜨며 작동을 멈춥니다.

---

## 2. 전력망 작동 원리 (Mechanics)

*   **Global Pool vs Local Grid**: 구현의 안정성을 위해 맵 전체의 발전량을 하나로 합산하는 **Global Pool** 방식을 사용하되, 건물이 전력을 받으려면 반드시 **활성화된 전력망(Power Node)의 반경 내**에 있어야만 하는 하이브리드 방식을 채택합니다.
*   **전력 우선순위 (Power Priority)**: 전력이 총 생산량보다 많이 필요할 경우 (블랙아웃 발생 시):
    1. 메인 서버 (Core) - 1순위 보호
    2. 방어 타워 - 2순위 (방어 최우선)
    3. 채굴기 및 가공소 - 3순위 (가동 중지)

---

## 3. 단계별 구현 계획 (Implementation Steps)

### Phase 1: 기본 전력 데이터 및 발전기 추가
- [ ] `config.js`에 전력 생산/소비량 데이터 추가. (예: Processor `-10W`, Power Plant `+50W`)
- [ ] 신규 건물 `Power Plant`(발전소) 및 `Power Node`(송신탑) 클래스 구현 및 UI 등록.
- [ ] 상단 HUD에 **전력 게이지 (예: `⚡ 45/100 W`)** 시각화 요소 추가.

### Phase 2: 전력망 계산 매니저 (PowerManager)
- [ ] `PowerManager.js` 생성 및 `MainScene` 연동.
- [ ] 매 틱(Tick)마다 메인 코어(Core)부터 시작해 서로 연결된 `Power Node`들을 탐색(BFS)하여 **활성화된 전력망 범위** 계산.
- [ ] 활성화된 발전기의 총 발전량 및 소비처의 총 소비량 계산.

### Phase 3: 건물의 전력 의존성 적용
- [ ] `BaseBuilding`에 `hasPower` 상태 플래그 추가.
- [ ] `TickSystem`에서 건물 동작 전에 '전력망 내부에 있는지' & '전력이 남았는지' 검사.
- [ ] 전력이 없는 건물의 경우 생산/가공 로직 일시 중지.

### Phase 4: 전력 UX 및 시각적 피드백
- [ ] `Power Node` 건설 시 파란색 반투명 오버레이로 **전력 공급 반경(Area)** 표시.
- [ ] 일반 건물을 지을 때(고스트 빌딩) 전력망이 닿는지 여부를 테두리 색상으로 피드백.
- [ ] 전력이 끊긴 건물 상단에 붉은색 번개(⚡) 아이콘 깜빡임 추가.

---

## 4. 플레이 경험 및 밸런스 기대 효과
*   초기에는 Core 자체가 약한 전력(예: 3x3 범위, 20W)을 제공하여 첫 채굴기를 지을 수 있게 유도합니다.
*   공장을 넓히기 위해서는 반드시 `Power Node`를 컨베이어 옆에 일정 간격으로 지으며 전력망을 확장해야 합니다. 이는 무분별한 맵 확장을 막고 더 계획적인 공장 레이아웃을 짜도록 만듭니다.
