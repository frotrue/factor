## Implementation Plans

The `superpowers/plans/` folder contains task-specific plans that were used to implement focused changes. These plans are useful for understanding why a feature was built, but the current docs above are the source of truth for present status.

## Archive

The `archive/` folder keeps older planning notes for traceability. Archived documents may describe ideas that were superseded, partially implemented, or renamed. Check [archive/README.md](archive/README.md) before treating any archived document as current guidance.
