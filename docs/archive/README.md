This folder contains historical planning notes from earlier phases of Neural Factory. These files are preserved for context, but they are not the current project source of truth.

Use the active documentation in `docs/` first:

- `docs/README.md` for the current documentation map.
- `docs/CONCEPT.md` and `docs/DIRECTION_REALIGNMENT_PLAN.md` for product direction.
- `docs/PROJECT_ANALYSIS_AND_ROADMAP.md` for status and roadmap.
- `docs/NEXT_TASKS.md` for immediate priorities.
- `docs/QA_CHECKLIST.md` and `docs/AUTOMATED_TESTING_GUIDE.md` for verification.

## Archive Sections

- `completed-plans/`: completed or superseded implementation plans.
- `diagnostics/`: point-in-time reviews, critiques, and diagnostic reports.
- `superpowers-plans/`: task-specific implementation plans generated for focused
  past changes.

Archived files may contain old terminology, incomplete plans, or implementation ideas that have since changed. When an archived note conflicts with an active document, follow the active document.
