# 🌐 Neural Factory: UI/UX 개선 및 고도화 계획서

> 모바일 개발 상태: 현재 모바일 개발은 일시 중단 상태입니다. 모바일 관련 구현, QA, 레이아웃 개선, 터치 조작 개선은 개발 재개 전까지 보류합니다.

이 문서는 **The Neural Factory**의 시각적 완성도와 사용자 경험(UX)을 향상시키기 위한 단계별 로드맵을 정의합니다.

---

## 📅 Phase A: 시각적 테마 및 HUD 리뉴얼 (Visual Identity)
**목표**: 사이버펑크 및 신경망 테마를 시각적으로 확립하고 핵심 정보를 전달합니다.

- [ ] **글래스모피즘(Glassmorphism) 적용**
  - UI 패널에 반투명 배경, 블러 효과, 네온 테두리 적용.
  - `rgba(0, 0, 0, 0.7)` 배경과 `backdrop-filter: blur(10px)` 사용.
- [ ] **네온 컬러 팔레트 시스템**
  - 자원별/상태별 네온 컬러 지정 (Cyan, Magenta, Volt Green).
  - 건물 및 아이템에 글로우(Glow) 효과 레이어 추가 검토.
- [ ] **상단 대시보드 (Global Stats)**
  - 시스템 신뢰도(Confidence Score) 진행 바.
  - 현재 에포크(Epoch) 및 시간 정보 표시.
- [ ] **커스텀 폰트 통합**
  - `JetBrains Mono` 또는 `Share Tech Mono` 등 코딩/테크니컬 폰트 적용.

---

## 🏗️ Phase B: 건설 UX 고도화 (Construction Interaction)
**목표**: 건물을 배치할 때의 직관성과 정확성을 높입니다.

- [ ] **고스트 빌딩 (Ghost Placement)**
  - 마우스 커서 위치에 배치될 건물의 반투명 실루엣(투명도 0.5) 표시.
  - 배치가 불가능한 지역일 경우 빨간색 색조 필터 적용.
- [ ] **회전 가이드 인디케이터**
  - 고스트 상태에서 출구 방향을 나타내는 밝은 화살표 표시.
  - `R`키를 누를 때마다 부드럽게 회전하는 애니메이션.
- [ ] **건설/철거 피드백 애니메이션**
  - 배치 시 건물이 튀어나오는 듯한 팝업 효과 (Tweening).
  - 철거 시 부서지는 파티클 효과 또는 페이드 아웃.

---

## ℹ️ Phase C: 건물 상태 정보 및 툴팁 (Information Design)
**목표**: 복잡해지는 공장 상황을 한눈에 파악할 수 있게 합니다.

- [ ] **실시간 인포 툴팁 (Hover Info)**
  - 건물 위에 마우스를 올리면 나타나는 상세 정보창.
  - 내부 인벤토리 상태 (예: `Data [■■□□□] 2/5`), 가공 속도 표시.
- [ ] **병목 현상 시각화 (Alert System)**
  - 출력구가 막혔거나 입력 재료가 부족할 때 건물 상단에 부유하는 아이콘 표시.
- [ ] **아이템 시각화 개선**
  - 단순한 원형에서 데이터 패킷 느낌의 사각형 또는 비트맵 스프라이트로 변경.

---

## ⚙️ Phase D: 테크니컬 및 편의성 강화 (Technical UX)
**목표**: 조작의 깊이를 더하고 시스템의 안정성을 높입니다.

- [ ] **단축키 시스템 및 UI 힌트**
  - 빌드 버튼 하단에 `[1]`, `[2]`, `[3]` 등 단축키 라벨 표시.
  - 단축키 입력 시 UI 버튼이 눌리는 시각적 효과.
- [ ] **알림 로그 (Activity Log)**
  - 화면 좌측 하단에 주요 이벤트(자원 고갈, 가공 완료 등)를 보여주는 작고 투명한 로그창.
- [ ] **설정 메뉴 (Settings)**
  - 게임 속도 조절 (1x, 2x, 4x).
  - 그리드 켜기/끄기 토글.

---

## 🚀 우선순위 로드맵

1. **Step 1**: Phase B (고스트 빌딩) - 게임플레이의 가장 직접적인 편의성 향상.
2. **Step 2**: Phase A (테마 리뉴얼) - 시각적 몰입도 확보.
3. **Step 3**: Phase C (툴팁 시스템) - 전략적 판단을 위한 정보 제공.
4. **Step 4**: Phase D (단축키 및 로그) - 숙련된 플레이를 위한 편의 기능.
